UnUTM
=====

A JavaScript library removes Urchin Traffic Monitor (UTM) parameters from URL.


INSTALL
-------

Install with Bower

    $ bower install unutm


USAGE
-----

Invoke this library by adding following HTML code to HTML:

    <script src="/bower_components/unutm/unutm.js"></script>
